Tableau Desktop — drag-and-drop data visualization, connect to any database, build interactive dashboards and share visual analytics across teams

### Features

- Connect to any data source live or extract
- Drag-and-drop dashboard builder
- Interactive filters and drill-down
- Map visualization with geocoding
- Publish to Tableau Server or Public

### Requirements

- Windows 10 / 11 (64-bit)
